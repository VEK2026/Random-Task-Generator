## Описание программы
Кратко: что делает программа, зачем она нужна, какие проблемы решает. Пример:
"Random Task Generator" - Генератор случайных задач.
GUI-приложение «Random Task Generator» с использованием библиотеки random, сохранением истории и Git.
Создает список предопределённых задач (например, «Прочитать статью», «Сделать зарядку») нажатием кнопки «Сгенерировать задачу», которая выбирает случайную задачу. Дополнительно отображает историю сгенерированных задач в виде списка. Реализуйет фильтрацию по типу задачи (учёба, спорт, работа). Сохраняет историю в JSON и загружает обратно. Проверяет корректность ввода.
##  Требования для запуска
Что нужно установить на компьютер: - Python 3.10 или выше - Библиотеки: pip install matplotlib requests ---
